## Mengapa Website Ini Dibangun?

Website ini adalah tempat saya bereksperimen dengan hal-hal baru, portfolio, serta untuk memperlancar kemampuan technical writing saya. Untuk sekarang, Website ini dibangun dengan [Next JS](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/), [Jotai](https://jotai.org/), [Shiki](https://shiki.matsu.io/), serta [MDX](https://mdxjs.com/). Selengkapnya, bisa langsung cek file [package.json](https://github.com/haikelz/haikel.my.id/blob/master/package.json) projectnya.
