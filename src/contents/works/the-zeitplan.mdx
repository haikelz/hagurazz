## Deskripsi

The Zeitplan adalah sebuah design yang dibuat oleh bang Nauval dari [Array ID](https://www.youtube.com/@arrayid). Disini saya menslicing design tersebut menjadi sebuah Website yang dapat diakses siapa saja.

Saya belajar banyak tentang Framer Motion dan memanfaatkan `useInView` hook untuk memunculkan animasi berdasarkan viewport. Jadi ketika first load atau direfresh page, ga langsung semua animasinya jalan.

## Tech Stack

- React JS
- Vite
- Typescript
- Tailwind CSS
- Framer Motion
