## Deskripsi

Devcode Gethired adalah sebuah challenge dari [gethired.id](https://gethired.id/blog/devcode-talent-hiring/). Disini saya mengambil challenge React JS, dengan studi kasus Todo List. Di challenge ini, peserta diharuskan mengikut aturan yang ada agar mendapatkan poin tinggi saat tahap testing.

Such a great experience bisa ikut challenge ini, karena saya juga sekalian mengimplementasikan beberapa teknologi yang jarang saya pakai seperti React Query.
