## Deskripsi

Wakaranai adalah sebuah Website sederhana untuk menggenerate Katakana dan Hiragana berdasarkan input romaji yang diketikkan oleh user. Selain itu, kita bisa menyimpan hasilnya dan melihatnya di page /result.

Untuk animasi, saya menggunakan library bernama `svelte-motion`(kalau di react, ini kayak framer-motion), dan routingnya saya menggunakan `svelte-navigator`.

## Tech Stack

- Svelte JS
- Tailwind CSS
