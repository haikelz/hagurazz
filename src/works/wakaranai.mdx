## Deskripsi

Wakaranai adalah sebuah Website sederhana untuk mengubah input romaji yang diberikan user ke Katakana dan Hiragana. Selain itu, user bisa menyimpan hasilnya dan melihatnya di page /result.

Untuk animasi, saya menggunakan library bernama `svelte-motion`(kalau di react, ini kayak framer-motion), dan routingnya saya menggunakan `svelte-navigator`.

## Tech Stack

- Svelte JS
- Tailwind CSS
