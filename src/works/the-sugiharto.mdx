## Description

The Sugiharto adalah sebuah design yang dibuat oleh bang Nauval dari [Array ID](https://www.youtube.com/@arrayid). Disini saya menslicing design tersebut menjadi sebuah Website yang dapat diakses siapa saja.

Pada slicing kali ini saya memanfaatkan Framer Motion untuk animasi, dan membuat beberapa page sesuai dengan design yang ada.

## Tech Stack

- Next JS
- Typescript
- Tailwind CSS
- Framer Motion
