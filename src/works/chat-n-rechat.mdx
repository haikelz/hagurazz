## Deskripsi

Chat n' rechat adalah sebuah design yang dibuat oleh bang Nauval dari [Array ID](https://www.youtube.com/@arrayid). Disini saya menslicing design tersebut menjadi sebuah Website yang dapat diakses siapa saja.

Pada slicing kali ini saya belajar tentang AOS untuk animasi, aksesibilitas dalam sebuah Website, dan memanfaatkan fitur baru di Next JS 13, yakni `appDir`.

Berikut [preview](https://chat-n-rechat-seven.vercel.app/) dan [repo githubnya](https://github.com/haikelz/chat-n-rechat).

## Tech Stack

- Next JS
- Typescript
- Tailwind CSS
- AOS
