Animeow adalah sebuah Website untuk mendapatkan informasi tentang anime dan manga. Data didapatkan dari [Jikan Moe API](https://jikan.moe/)

Disini saya belajar tentang pemanfaatan `useMemo`, juga React Query untuk fetching data.

Berikut [preview](https://animeow-red.vercel.app/) dan [repo githubnya](https://github.com/haikelz/animeow).

## Tech Stack

- Next JS
- Typescript
- Tailwind CSS
- React Query
- Chakra UI
- Jotai
